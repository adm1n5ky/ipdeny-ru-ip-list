# ipdeny-ru-ip-list

Автоматически обновляемые списки IPv4 и IPv6 адресов Российской Федерации (RU).

## Источник данных
Данные агрегируются и загружаются с сайта [IPdeny.com](https://ipdeny.com).
*По требованию правообладателя:* **Powered by IPDENY.COM IP database.**

## Как это работает
Скрипт GitHub Actions запускается каждый день в 00:00 UTC, проверяет наличие обновлений на стороне IPdeny и, если базы изменились, автоматически обновляет файлы в этом репозитории.

## Лицензия и копирайт
Оригинальные права на списки принадлежат IPdeny и региональным интернет-регистраторам (RIPE, APNIC и др.). Полный текст условий использования находится в файле `Copyrights.txt`. Данный репозиторий является зеркалом общего пользования.
